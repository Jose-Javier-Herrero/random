## Random para node.js


Este repositorio contiene el proyecto **Random** de [jquemada](https://github.com/jquemada), que podéis encontrar en la dirección [https://github.com/jquemada/random](https://github.com/jquemada/random). 
Éste forma parte de un ejercicio del curso de desarrollo de servicios en la nube con HTML5, Javascript y node.js, que podéis encontrar en el siguiente enlace: [Míriada X] (https://www.miriadax.net/w1eb/javascript-node-js)



### Descripción del proyecto


Este proyecto contiene un programa para [node.js] (https://nodejs.org/) que genera un número aleatorio de 0 a 1 y que indica por consola si el número generado es mayor o menor que 0,5. 


### Cambios respecto al proyecto original


Los cambios respecto a la versión original de [jquemada](https://github.com/jquemada) son los siguientes:
* Añadido el fichero de licencia MIT
* Añadido el fichero readme al que estáis accediendo


### Instalación


No es necesario instalar este programa, no obstante, es necesaria la instalación de [node.js] (https://nodejs.org/).



### Utilitzación


Para ejecutar el programa, tenéis que ejecutar el JS mediante el comando node. 

    `node random.js`
